Requirements and dependencies:
    1. random-->randint
    2. time
    3. Math
    4.matplotlib.pyplot


Running instructions:

	run the following commands:

		1)for Insertion sort 			:	Insertion.py
		2)for Selection sort 			:	Selection.py
		3)for Bubble sort 			:	Bubble.py
		4)for Merge sort 				:	Mergesort.py
		5)for Heap sort 				:	Heapsort.py
		6)for quick sort 				:	quicksort.py
		7)for quick sort(median)		:	Quicksort_median.py
		8)for bar graph of run time		:	barplot.py (This file ask user to enter size,lowest and highest number for all algorithmns then bar plot opens) 	

	Instructions after clicked on run:
		1)enter size of the list in the console
		2)enter lowest number in the list in the console
		3)enter highest number in the list in the console
